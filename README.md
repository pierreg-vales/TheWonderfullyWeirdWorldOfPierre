# TheWonderfullyWeirdWorldOfPierre
Interactive Portofolio To Showcase all aspects of Pierre's life. Have fun exploring my hobbies, likes, interests, academics, experience, etc in this game-like world. 
Includes recruiter mode that showcases academics and profesional experience in an easy to understand way. 

## Installation

**1. Clone the repository**
git clone https://github.com/pierreg-vales/TheWonderfullyWeirdWorldOfPierre.git
cd TheWonderfullyWeirdWorldOfPierre

**2. Create a virtual environment**

Windows:
python -m venv venv

Mac/Linux:
python3 -m venv venv

**3. Activate the virtual environment**

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

**4. Install dependencies**
pip install -r requirements.txt

## Usage
python app.py
